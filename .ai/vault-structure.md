# Vault Structure

This document outlines the recommended folder structure and organization for your UPSC Obsidian Vault. Following this structure will help maintain clarity, ease of navigation, and efficient note-taking.

```
root/
│ .gitignore
│ LICENSE
│ README.md
│
├───.ai/ # Rules, auto-generation logic for IDE/AI tools
│ rules.md
│
├───.obsidian/ # Obsidian settings (plugins, graph config, layout)
│
├───00-Index/ # Master Tables of Content & High-level Maps
│ │ MOC-Current-Affairs.md
│ │ MOC-Economy.md
│ │ MOC-Environment.md
│ │ MOC-Geography.md
│ │ MOC-GS.md
│ │ MOC-History.md
│ │ MOC-Polity.md
│ │ README.md
│ │
│ └───_templates/
│ \_template-ca.md
│ \_template-moc.md
│ \_template-topic.md
│
├───10-Static/ # Atomic topic notes (NCERT, Std Books)
│ Geography-Atmosphere.md
│
├───20-Current-Affairs/ # Date-prefixed CA notes
│ README.md
│
├───30-Optional/ # Optional subject notes
│ README.md
│
├───40-Practice/ # Answer writing, PYQs, mock analysis
│ README.md
│
└───assets/ # PDFs, images, charts, external refs
README.md

```
